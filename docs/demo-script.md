# AgentWire Demo Video Script

> Tone: Deadpan, self-deprecating, irreverent. Think Dollar Shave Club meets informercial parody.

---

## INTRO

**"Are you FAT? LAZY? Moderately intelligent?"**

*[You sitting in chair, barely turning to camera]*

**"If you said yes to at least ONE of these things... AgentWire was built for you."**

*[Gesture vaguely at screen]*

"I didn't want to TYPE anymore. My fingers were TIRED."

*[Hold up phone]*

"Now I just... talk. From wherever I am."

*[Press PTT from couch]*
"Hey, add error handling to that function."

*[Agent starts typing furiously]*

"See? It's working. I'm not."

## MULTI-SESSION

**"But I have MULTIPLE projects!"**

"Cool. So do I. I'm lazy, not unambitious."

*[Show dashboard with multiple sessions]*

"They all run at once. I supervise. Loosely."
