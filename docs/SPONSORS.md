## Notes for Setup

**Tier limits:** GitHub allows up to 10 tiers total (we have exactly 10).

**Custom amounts:** GitHub also allows sponsors to enter custom amounts — you don't need a tier for that.

**What to avoid promising:**
- Hard SLAs (use "priority" and "commitment" instead of "guarantee")
- Unlimited anything
- Features that don't exist yet
- Time-based commitments you can't keep

**Fulfillment checklist:**
- [ ] Create SPONSORS.md file for listing names
- [ ] Set up Discord with sponsor roles
- [ ] Create a way to track logo placement durations
- [ ] Calendar system for support calls
